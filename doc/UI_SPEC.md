# UI 規格（初版 — BNI 紅主題）

> 本規格為初版占位，待您提供正式設計圖後，以設計稿為準更新色彩、字體與間距 token。

## 1. 設計原則

- **現場優先**：大轉盤、高對比、少步驟。
- **BNI 識別**：以紅色為主，搭配米白 / 金色點綴。
- **不打斷節奏**：抽獎中不需操作左側；換獎項時再編輯。

## 2. 色彩 Token

| Token | 色碼 | 用途 |
|-------|------|------|
| `--bni-red` | `#C8102E` | 主色：標題、按鈕、轉盤邊框 |
| `--bni-red-dark` | `#8B0000` | 深紅：hover、強調 |
| `--bni-cream` | `#F5F0E6` | 米白：卡片背景、扇區交替 |
| `--bni-gold` | `#C9A227` | 金色：指針、裝飾線 |
| `--surface` | `#FAFAFA` | 頁面背景 |
| `--text` | `#1A1A1A` | 內文 |

## 3. 字體

- 標題：系統無襯線粗體（`font-bold`）
- 內文：系統 UI 字體堆疊
- 轉盤扇區：`clamp(0.65rem, 1.8vmin, 0.95rem)`

## 4. 版面結構（桌面 ≥ 1024px）

```
┌────────────────────────────────────────────────────────────┐
│  BNI Lucky Draw                    [音效] [重置得獎]      │
│  僅本次瀏覽有效，關閉分頁即重置                              │
├──────────────────────┬─────────────────────────────────────┤
│  抽獎項目             │                                     │
│  ┌────────────────┐  │         ╭─────────────╮           │
│  │   圖片上傳區    │  │         │   轉 盤     │  ◀ 指針   │
│  └────────────────┘  │         ╰─────────────╯           │
│  [項目標題輸入框]      │         [ 開始抽獎 ]                │
├──────────────────────┴─────────────────────────────────────┤
│  項目（空格/換行）    │  名單（空格/換行）                   │
│  [ textarea ]        │  [ textarea ]                       │
├────────────────────────────────────────────────────────────┤
│              ☕ 請我喝杯咖啡 →                              │
└────────────────────────────────────────────────────────────┘
```

### 行動裝置（< 1024px）

- 順序：頂部列 → 項目面板 → 轉盤 → 雙欄輸入（堆疊）→ 頁尾。
- 轉盤寬度：`min(90vw, 420px)`。

## 5. 元件規格

### 5.1 PrizePanel（左側項目）

| 屬性 | 規格 |
|------|------|
| 圖片區 | 比例 4:3，`border-2 border-dashed`，hover 顯示「點擊上傳」 |
| 上傳 | `accept="image/*"`，最大 2MB |
| 標題 | 單行 `input`，placeholder「輸入抽獎項目名稱」 |
| 邊框 | 2px solid `--bni-red` |

### 5.2 LotteryWheel（轉盤）

| 屬性 | 規格 |
|------|------|
| 外形 | 正圓，外框 6px `--bni-red` |
| 扇區 | 交替 `--bni-cream` / `#FFFFFF`，文字 `--text` |
| 指針 | 頂部三角形，`--bni-gold` + 陰影 |
| 中心按鈕 | 圓形，`--bni-red`，白字「開始抽獎」 |
| 旋轉中 | 按鈕 disabled，opacity 0.7 |
| 空名單 | 灰色占位圓 + 說明文字 |

### 5.3 WinnerModal

| 屬性 | 規格 |
|------|------|
| 遮罩 | `rgba(0,0,0,0.6)` |
| 卡片 | 白底、圓角 16px、紅色頂條 4px |
| 標題 | 「恭喜得獎！」 |
| 內容 | 得獎者大字、項目名小字 |
| 按鈕 | 「繼續抽獎」`--bni-red` |
| 動效 | 入場 scale 0.9 → 1（CSS） |

### 5.4 DualTextInput

| 欄位 | Label | Placeholder |
|------|-------|-------------|
| 左 | 抽獎項目 | 每行或空格分隔多個項目名稱… |
| 右 | 抽獎名單 | 每行或空格分隔參與者姓名… |

- 最小高度：120px
- 聚焦：紅色 ring

### 5.5 SponsorFooter

- 置中連結，咖啡圖示 +「請我喝杯咖啡」
- `target="_blank"` `rel="noopener noreferrer"`

## 6. 互動狀態

| 狀態 | 視覺 |
|------|------|
| 預設 | 可點擊轉盤 |
| 旋轉中 | 轉盤旋轉、按鈕 disabled |
| 得獎 | Modal + confetti |
| 無候選 | 按鈕 disabled + toast 式提示 |

## 7. 無障礙（基本）

- 按鈕具 `aria-label`
- Modal 開啟時焦點移至關閉按鈕
- 色彩對比：紅底白字按鈕對比度 ≥ 4.5:1
