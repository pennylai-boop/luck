## 使用者操作流程

### 例會前準備

1. 開啟網頁，確認頂部提示「資料僅保留於本次瀏覽」。
2. 在**右下名單欄**貼上或輸入參與者（空格或換行分隔）。
3. 在**左下項目欄**可選填項目名稱備忘。
4. 在**左側面板**上傳本輪獎品圖片，輸入項目標題（例如「本月引荐王」）。

### 現場抽獎

1. 全螢幕投影，確認轉盤扇區與名單一致。
2. 點擊轉盤中心「開始抽獎」。
3. 轉盤減速停止 → 彈窗顯示得獎者 → 煙火（若開啟音效則播放）。
4. 關閉彈窗；得獎者已自名單候選中排除。
5. 若要抽下一個獎項：更新左側圖片與標題，再點轉盤。
6. 重複直到名單抽完；必要時點「重置得獎紀錄」重新抽。

### 換場 / 結束

- 關閉分頁即清空，下一場例會重新輸入即可。

## 畫面區塊對照

| 區塊 | 用途 |
|------|------|
| 頂部列 | 標題、音效開關、重置得獎 |
| 左上 | 目前抽獎項目（圖 + 文） |
| 右上 | 幸運轉盤 |
| 左下輸入 | 項目文字清單（選用） |
| 右下輸入 | 名單（轉盤資料來源） |
| 頁尾 | 贊助連結 |
